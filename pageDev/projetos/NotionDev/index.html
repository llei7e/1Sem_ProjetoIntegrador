O que aprendi:

Histórico

• Introduzido com o clássico "Hello World", aprendemos que páginas web rodam no lado do cliente e são constituidas de HTML, CSS e JavaScript.

• Logo após conhecemos a semântica em HTML que consiste no uso de elementos que tem um função, significado ou conteúdo específico. A utilização desses elementos ajudam a estruturar a página web de forma mais ágil.

• De forma a aplicar o HTML semântico, utilizando CSS cada vez com mais recursos, e introduzir o uso de JavaScript criamos a página web que recebe Peso e Altura de uma pessoa e realiza o cálculo de IMC.

• Com instruções e acompanhamento do professor Gabriel, tivemos um desenvolvimento numa parte mais focada num site de login, utilizando tags, e funções de css e JavaScript diferentes, como inserção de imagens, redirecionamento para outras páginas, condições em JavaScript limitando login/password de usuários.

• Ja na criação do ToDo foi reforçado o HTML semântico e aprendendo novas funções css.

• Dando seguimento as aulas do professor Andre nos foi solicitado desenvolver uma calculadora, focando no HTML e CSS.

• Com a calculadora criada, o próximo passo era gerar funcionalidades para essa calculadora, inicialmente operações mátemáticas básicas, e posteriormente funções como exponenciação, raciação, calculo posfixo alem do infixo. (O que acabou sendo mais complicado do que todos esperávamos).

• Durante o andamento da matéria, conhecemos a árvore DOM e como criar e estilizar elementos a partir do JavaScript tornando nossos códigos mais flexíveis.

• De forma a nos avaliar, nos foi solcitado fazer uma página web contendo os seguintes projetos num iFrame: Calculadora INSS, um gerador de scripts (que inicialmente estva muito acima das minhas capacidades) e o projeto de um formulário no qual recebiamos informações atraves de 3 campos, e ao enviar o formulário, era adicionado a uma tabela abaixo na mesma página web.
